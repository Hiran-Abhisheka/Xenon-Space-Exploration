# 🚀 Xenon Space Exploration

A cinematic space exploration web experience built with **React**, featuring immersive animations, futuristic UI, and smooth interactive transitions.

🔗 **Live Demo:** https://xenon-space-exploration.vercel.app/  
📂 **Repository:** https://github.com/Hiran-Abhisheka/Xenon-Space-Exploration

## ✨ Features

- 🎬 Cinematic animations with smooth scroll-based transitions  
- 🚪 Interactive UI elements and dynamic visual effects  
- 🌑 Futuristic dark-themed design  
- ⚛️ Built with reusable React components  
- ⚡ Optimized for performance and smooth rendering  
- 📱 Fully responsive (mobile + desktop)  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** CSS3 (animations, transitions)  
- **Deployment:** Vercel  
- **Version Control:** Git & GitHub

## 🚀 Getting Started

### Clone the repository
```bash
git clone https://github.com/Hiran-Abhisheka/Xenon-Space-Exploration.git
cd Xenon-Space-Exploration
```

### Install dependencies
```bash
npm install
```

### Run the project
```bash
npm run dev
```
